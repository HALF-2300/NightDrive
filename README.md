# NightDrive

Premium car dealership platform — real-time MarketCheck inventory, lead capture, and production-ready security.

**By [HALF-2300](https://github.com/HALF-2300)**

## Quick start

```bash
npm install
cp .env.example .env
# Edit .env: set MARKETCHECK_API_KEY (required for inventory)
npm start
```

App runs at **http://localhost:1335**.

## Push to your GitHub

1. On [GitHub](https://github.com/new), create a new repository (e.g. `nightdrive` or `halfcar`).
2. Then run:

```bash
git add .
git commit -m "Initial commit: NightDrive"
git branch -M main
git remote add origin https://github.com/HALF-2300/YOUR-REPO-NAME.git
git push -u origin main
```

Replace `YOUR-REPO-NAME` with the repo you created.
